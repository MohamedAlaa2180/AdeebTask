Pro Scene Manager - Documentation

Overview

Pro Scene Manager is a Unity Editor tool that simplifies scene management, allowing you to organize, load, and manage scenes efficiently.

Installation

  1. Import the Pro Scene Manager package into your Unity project.
  2. Open the tool via Tools -> Pro Scene Manager in the Unity Editor.

Features

  • Automatically lists all scenes in the project.
  • Categorizes scenes based on folder structure.
  • Quickly load scenes with a single click.
  • Supports additive scene loading.
  • Allows adding personal notes to scenes.
  • Indicates currently loaded scenes.
  • Search functionality to find scenes easily.
  • Scene selection feature in the Project window.
  • Refresh button to update the scene list.

Usage

Loading Scenes

Click the Load button to open a scene. Use the Additive checkbox to load a scene additively.

Managing Notes

Click the Notes button to open the notes editor. If a scene has notes, the button will change appearance.

Searching Scenes

Use the search bar at the top of the window to filter scenes by name.

Selecting Scene Asset

Click the Select button to highlight the scene in the Project window.

Compatibility

Pro Scene Manager is developed with Unity 2021.3.0f1 It is recommended to use a Unity version that supports the latest editor UI features.
